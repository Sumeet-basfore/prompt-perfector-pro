# Tech Stack & Tools

## Stack Overview
| Layer | Technology | Why |
|-------|-----------|-----|
| Frontend | React + Tailwind CSS | Generated by Lovable; clean, exportable, mobile-first |
| Build Tool | Lovable.dev (primary) / Bolt.new (backup) | Fastest path to working app for vibe-coders |
| Hosting | Vercel (free tier) | One-click deploy from Lovable, automatic HTTPS, 100GB bandwidth |
| Database | Supabase (free tier) | Built into Lovable, handles both DB and optional auth |
| AI Engine | Gemini 2.5 Flash (primary) | Free: 250 req/day — enough for first 200 DAU |
| AI Fallback | Groq / Llama 3.3 70B | Free: ~14,400 req/day — handles spikes automatically |
| Analytics | Umami (free cloud) | Privacy-first, no cookie banners, free |

## API Endpoints

### Gemini (Primary)
```
URL: https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent
Method: POST
Auth: ?key=GEMINI_API_KEY (query param)
```

### Groq (Fallback — triggered on Gemini 429/503)
```
URL: https://api.groq.com/openai/v1/chat/completions
Method: POST
Auth: Bearer GROQ_API_KEY
Model: llama-3.3-70b-versatile
```

## Typography
- Font: **Inter** (Google Fonts — include via @import in CSS)
- Headline: 32px / weight 700
- Body: 16px / weight 400
- Helper text: 14px / weight 400 / color: text-secondary

## Layout Rules
- Rounded corners: 12px (rounded-xl in Tailwind)
- Spacing unit: 4 (use p-4, gap-4, m-4 as base)
- Mobile breakpoint: 375px (iPhone SE — design for this as minimum)
- Stack vertically below 640px (sm: breakpoint in Tailwind)

## Supabase Schema
Run this SQL in Supabase dashboard → SQL Editor:

```sql
-- Track transformations (no PII — character counts only, not actual text)
CREATE TABLE transformations (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  created_at TIMESTAMP DEFAULT NOW(),
  template_category TEXT,
  ai_editor TEXT,
  model_used TEXT,         -- 'gemini' or 'groq'
  input_length INTEGER,    -- Character count only
  output_length INTEGER,
  was_copied BOOLEAN DEFAULT FALSE,
  feedback TEXT            -- 'useful' | 'not_useful' | NULL
);

-- Monitor API fallbacks
CREATE TABLE api_events (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  created_at TIMESTAMP DEFAULT NOW(),
  model_attempted TEXT,
  success BOOLEAN,
  error_code INTEGER
);

-- Row Level Security: allow anonymous inserts, no client reads
ALTER TABLE transformations ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Allow anon inserts" ON transformations 
  FOR INSERT WITH CHECK (true);
```
